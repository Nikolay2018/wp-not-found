== Description ==

Plugin to logging Not Found errors to local file on your server. This can be useful for tracking down broken links or missing content. You can optionally record additional information such as IP addresses, user agents and referrers.

== Installation ==

1. Upload the plugin folder to the '/wp-content/plugins/' directory.
2. Activate the 'WP Not Found' plugin through the 'Plugins' menu in WordPress.
3. The plugin settings can be accessed via the 'WP Not Found' menu in the administration area.
